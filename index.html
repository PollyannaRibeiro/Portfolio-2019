WordPress Blog Theme

This is my second website created in WordPress. The first was the years ago, without any knowledge of the language and simply created to keep up my blog. This time, I used my knowledge in JS, JQuery, I studied a little bit about how WordPress works, studying Codex. And I used Sass for the first time, which already gave me a very clean and readable code, pretty different than the previous one. This site works properly on a mobile, tablet or computer, it's responsive and easy to use.

This is a blog aimed for Brazilians tourists that want to understand better London before coming to visit the city.

responsive

Depending on the size of the device screen, it will show all the highlight content or will remove some of them.

I used Bootstrap to create a card column style that changes the number of columns, depending on the size of the screen.

Articles and other pages, also are responsive and some of them including a sidebar with ads from Google.

All the design of this site, including the logo and other details, were created by me.

React - Neighborhood Map

This project was the last one created on the Web development Udacity Nanodegree course.

I had to use Google Map API, Wikipedia API and React to build from scratch a free theme App. I chose to do an App that shows the Museums options in a part of London.

After clicking on an option, it will animate the map icon and show the content from Wikipedia, including the name of the Museum, address, phone, rating, picture and a definition about the respective place.

This is a responsive and accessible web site. Using ARIA attributes, I built a website that provides context for screen readers.

Ajax with Fetch App

This is an asynchronous request App made using AJAX with fetch. The two APIs used are:

  • The NY Times - to collect the articles
  • Unsplash - to get the pictures

The project

Before informing what the users are interested today, they can see the most recent articles of the NY Times.

After submitting a request, they can see if it returns some picture and articles about the subject that they are interested in.

I did not use any different style from the offered by Udacity, so this project was only to evaluate my skills working on JS, HTML and CSS, and not my design skills. It's also a responsive app, I used Bootstrap cards to create responsive style.

Works with promises was a big challenge for me. It was like discovering a new world of possibilities that I've never worked before.
